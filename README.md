# Virtual-Lab-Setup-using-CentOS-Linux-
Setup of CentOS virtual machine using VirtualBox with Linux commands and Git basics

# Project Overview
This project demonstrates the setup of a virtual lab environment using CentOS Stream 9 on Oracle VirtualBox. It includes system installation, configuration, and basic Linux administration.

# What I Did
- Installed CentOS Stream 9 on Oracle VirtualBox  
- Created and managed virtual machines  
- Configured system settings and performed troubleshooting  
- Used Linux commands for system administration ( pwd, cd, mkdir, ls, vagrant status, vagrant ssh, whoami , sudo -i, exit )
- Practiced Git commands for version control  

# Tools & Technologies
- Oracle VirtualBox  
- CentOS Stream 9  
- Linux Commands  
- Git  

# Key Learnings
- Virtualization concepts and VM setup  
- Linux system administration basics  
- Cloud and DevOps fundamentals  
- Hands-on troubleshooting skills
